Заготовка конфигурации для Nginx.

Использовать обдуманно, чтобы не нарушить работу сервера Nginx!
Иными словами, не нужно тупо копи-пастить, договорились!? ;)

nginx.conf - основная конфигурация Nginx.

default - для хостов, которые должны отвечать на запросы к IP http://1.2.3.4/

vhosts/site.ru.conf - конфигурация сайта с названием доменов.

vhosts/site.ru - основная конфигурация сайта.
